2. Naming Conventions

Module 02
The Golden Rule of File Naming

Use lowercase only. Use hyphens instead of spaces. Never use underscores. If the file name doesn't tell you what it is, what year it's for, and its version—rename it before saving.

6. SEO & CMS Workflow

Module 06 & 07

The 5-Step SEO Workflow

  1. Clean Page URL/Item : Use accurate, hyphenated page names (e.g., /shows/chronicles-of-the-highwaymen).
  2. Meta Tags: Complete the SEO Title and Description for every new page in the Digital Allies command center.
  3. Alt Text: Add descriptive alt text (e.g., "Cast of The Importance of Being Earnest performing on stage at Beale Street Theater in Kingman Arizona.")
  4. Structured Data: Ensure JSON-LD Event Schema is attached to every show page.
  5. Social Previews: Confirm the OG image is set so links look great on Facebook.

The Waterfall Effect (Publishing)

Always edit on the Desktop view first. Changes cascade down to tablet and mobile. Mobile-only tweaks do not affect desktop, but desktop changes will overwrite mobile if done incorrectly. Click Republish ONLY after all metadata and responsive views are confirmed.
