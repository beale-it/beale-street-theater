## 🎨 Core Theme Files

### `bst-theme.css`
**What it is:** Complete CSS design system with all styles  
**File size:** ~800 lines  
**Where to use:** Duda **Custom CSS**

**Contains:**
- CSS variables (colors, fonts, spacing)
- Typography scales (headings, body text, labels)
- Button system (3 variants × 3 sizes)
- Card components with BST Curve
- Grid system (responsive 1-4 columns)
- Navigation styles
- Section headers
- Utility classes
- Responsive breakpoints

**How to use:**
1. Copy entire file
2. Paste into Duda → Design → Custom CSS
3. Save

---

### `bst-scripts.js`
**What it is:** Vanilla JavaScript for interactive features  
**File size:** ~400 lines  
**Where to use:** Duda **Custom Code → Footer**

**Contains:**
- Mobile navigation toggle
- Active link highlighting
- Smooth scroll
- Card hover effects
- Lazy load images
- Show card click handlers
- Newsletter form handler
- Event calendar filter
- Scroll reveal animations
- Sticky navbar

**How to use:**
1. Go to Site Settings → Custom Code → Footer
2. Paste between `<script>` tags
3. Save

## 🎯 Quick Task Guide

### "I want to create a homepage"
1. Read: `DUDA-SETUP-GUIDE.md` (Steps 1-3)
2. Use: `pages/home-page.html` (paste entire file)
3. Customize images and text

---

### "I want to add a button"
1. Open: `snippets/all-buttons.html`
2. Find button type you need
3. Copy/paste into your page
4. Update `href` and text

---

### "I want to add a hero section"
1. Open: `snippets/hero-sections.html`
2. Choose layout (10 options)
3. Copy/paste
4. Replace text/images/links

---

### "I want to create a show listing page"
1. Open: `snippets/hero-sections.html` (copy "Compact Hero")
2. Open: `snippets/all-section-headings.html` (copy "Standard Section Header")
3. Open: `snippets/show-cards.html` (copy "Show Cards Grid")
4. Stack them in order
5. Customize content

---

### "I want to add icons"
1. Open: `snippets/all-icons.html`
2. Browse icon showcase
3. Copy SVG code
4. Paste into your component
5. Adjust `width`, `height`, `stroke` color

---

### "I want to customize colors"
1. Read: `DESIGN-TOKENS.md` (Color Palette section)
2. Edit: Custom CSS in Duda
3. Change `:root` variables
4. Save (all components update automatically)

---

### "I want to add a footer"
1. Open: `component-templates.html`
2. Find "FOOTER" section
3. Copy entire footer code
4. Paste at bottom of page
5. Update links and contact info

## 🎨 Design System Quick Reference

### Colors
- Stage Black: `#111315`
- Neon Teal: `#14b8a6`
- Text Primary: `#e8e8e8`
- Text Secondary: `#8a8f98`

### Fonts
- Headings: League Spartan (Bold, Uppercase)
- Body: Roboto (Regular)
- Buttons: Noto Sans (Bold, Uppercase)

### BST Curve
- Border-radius: `0px 10px 10px 0px`

### Spacing
- XS: 8px
- SM: 16px
- MD: 24px
- LG: 32px
- XL: 48px
